# ClawCommunication UI/UX 改进设计规格

## 1. 配色方案

### 调色板 (Dark Theme)
```css
:root {
  /* 背景色 - 渐变层次 */
  --bg-primary: #0f172a;        /* 主背景：深海军蓝 */
  --bg-secondary: #1e293b;      /* 卡片/面板背景 */
  --bg-tertiary: #334155;       /* 输入框/次级区域 */
  --bg-hover: #475569;          /* Hover 状态 */

  /* 强调色 */
  --accent-primary: #6366f1;    /* 主强调：靛蓝色 */
  --accent-secondary: #8b5cf6;  /* 次强调：紫色 */
  --accent-success: #22c55e;    /* 成功：翠绿 */
  --accent-warning: #f59e0b;    /* 警告：琥珀 */
  --accent-error: #ef4444;      /* 错误：红 */

  /* 文字色 */
  --text-primary: #f8fafc;      /* 主文字 */
  --text-secondary: #94a3b8;    /* 次要文字 */
  --text-muted: #64748b;        /* 弱化文字 */

  /* 边框 */
  --border-color: #334155;
  --border-light: #1e293b;
}
```

### AI 状态指示色
```css
.status-online { background: #22c55e; box-shadow: 0 0 8px #22c55e80; }
.status-away { background: #f59e0b; box-shadow: 0 0 8px #f59e0b80; }
.status-offline { background: #ef4444; box-shadow: 0 0 8px #ef444480; }
.status-processing { 
  background: linear-gradient(90deg, #6366f1, #8b5cf6);
  animation: pulse 1.5s ease-in-out infinite;
}
```

## 3. 布局结构

```
┌─────────────────────────────────────────────────────┐

├─────────────────────────────────────────────────────┤
│  Tab Navigation (48px)                              │

├─────────────────────────────────────────────────────┤
│                                                     │
│  Main Content Area                                 │

│                                                     │
│  ┌─────────────────────────────────────────────┐   │
│  │  Panel Container                            │   │

│  └─────────────────────────────────────────────┘   │
```

### 响应式断点
- Mobile: < 640px (单列布局)
- Tablet: 640px - 1024px (自适应)
- Desktop: > 1024px (最大宽度 1200px)

## 4. 组件设计

### 4.1 消息气泡 (Message Bubble)

```
┌─────────────────────────────────────────────┐
│  [头像]  用户名            时间 状态标签    │
│         消息内容                              │
│         处理状态                              │
└─────────────────────────────────────────────┘
```

**样式细节：**
- 头像：40x40px，圆角 50% (圆形)
- 气泡内边距：12px 16px
- 背景：bg-secondary
- 边框-radius：12px (左上是圆角，其他直角)
- Hover：轻微上移 + 背景变亮 + 阴影

**入站消息** (来自他人)：左侧对齐
**出站消息** (来自自己)：右侧对齐 (可选)

### 4.2 AI 状态卡片 (Node Status Card)

```
┌──────────────────────────┐
│  🟢  Wisp               │  ← 状态指示灯 + 名称
│  ─────────────────────── │
│  端口: 18789            │  ← 端口信息
│  状态: 在线             │  ← 状态文字
│  最近活跃: 2分钟前      │  ← 时间信息
│                          │
│  [查看详情]             │  ← 操作按钮
└──────────────────────────┘
```

**卡片样式：**
- 尺寸：min-width: 220px
- 背景：bg-secondary
- 边框：1px solid border-color
- 边框-radius：16px
- 阴影：0 4px 6px -1px rgba(0,0,0,0.3)
- Hover：transform: translateY(-2px) + 阴影加深

### 4.3 任务看板 (Kanban Board)

```
┌──────────┬──────────┬──────────┬──────────┐
│ 待处理    │ 进行中   │ 已完成   │ 失败     │
├──────────┼──────────┼──────────┼──────────┤
│ ┌──────┐ │ ┌──────┐ │ ┌──────┐ │ ┌──────┐ │
│ │任务卡 │ │ │任务卡 │ │ │任务卡 │ │ │任务卡 │ │
│ └──────┘ │ └──────┘ │ └──────┘ │ └──────┘ │
│ ┌──────┐ │ ┌──────┐ │          │          │
│ │任务卡 │ │ │任务卡 │ │          │          │
│ └──────┘ │ └──────┘ │          │          │
└──────────┴──────────┴──────────┴──────────┘
```

**列样式：**
- 背景：bg-tertiary (半透明)
- 列标题栏：bg-secondary
- 列标题颜色：对应状态色
- 卡片拖拽视觉反馈

**任务卡片样式：**
- 背景：bg-secondary
- 左边框：4px solid (状态色)
- 标题：text-base, font-medium
- 标签：badge 样式 (负责人、优先级)
- Hover：轻微上浮

## 5. 交互动效

### 全局过渡
```css
* {
  transition: all 0.2s ease;
}
```

### 按钮 Hover
- 背景色变亮 10%
- 轻微 scale(1.02)
- 0.15s 过渡

### 卡片 Hover
- translateY(-4px)
- 阴影加深
- 边框颜色变亮

### 消息气泡
- 新消息滑入：slideInLeft / slideInRight
- 脉冲动画：状态处理中时

### Tab 切换
- 下划线滑动动画
- 面板淡入淡出

### 状态指示灯
- 在线：呼吸灯效果 (脉冲)
- 处理中：旋转或流动渐变
- 离线：静态
